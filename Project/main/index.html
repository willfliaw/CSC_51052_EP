Olympics Dataset Visualisation

This plot is a directed chord diagram that visualizes the migration flows of athletes between their birth countries and the countries they represented in the Olympics.
This dynamic choropleth map visualizes various Olympic-related data, such as the number of Olympic debuts, hosting occurrences, and medal counts, by country.
This plot visualizes Olympic rivalries as a network, where each node represents a country. The user can adjust link distance to explore relationships. Selected and top rival countries are highlighted for clarity.
This plot displays the cumulative Olympic medals won by selected countries. Each line represents a country's total medal count over time, with an interactive "play" feature to observe medal progression dynamically by year.
This plot shows the Olympic medal density by country over time on a world map. A time slider at the bottom allows users to observe changes in medal distribution over specific years, highlighting the global spread of Olympic success.
The plot illustrates Olympic participation trends over time, showing the number of participating countries. The line graph highlights significant dips and increases, with key highlighted key events.
